# Pentomino-Pandemonium
A game made with the intention to learn using GameMaker.
Pentomino Pandemonium

A modern implementation of the classical Pentomino puzzle game developed in GameMaker as part of the M.Sc. Computer Animation and Game Technologies program at Hacettepe University.

Overview

Pentomino Pandemonium is a puzzle game where players solve increasingly challenging pentomino tiling problems by placing, rotating and flipping puzzle pieces on uniquely shaped grids.

The project focuses on gameplay programming, interaction systems, level architecture and technical game design rather than graphical complexity.

Features
16 handcrafted puzzle levels
Grid-based drag & drop interaction
Piece rotation (90° increments)
Piece reflection (mirror mechanics)
Collision detection
Grid snapping
Placement validation
Puzzle completion detection
Level progression
Main Menu
Credits Menu
Debug Level Skip mode

Gameplay

Players must cover every tile on the board using all available pentomino pieces without overlapping or leaving empty spaces.

Each puzzle has multiple valid solutions while increasing gradually in complexity.
